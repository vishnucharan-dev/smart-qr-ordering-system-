# smart-qr-ordering-system-
# 🍽 TableFlow QR Ordering System

> A QR-based restaurant ordering platform built using **n8n** to automate customer ordering, kitchen notifications, and order logging.

---

# 📖 Project Overview

TableFlow is a digital ordering system that allows restaurant customers to scan a QR code, browse the menu, and place orders directly from their mobile devices.

The system automates order processing by instantly notifying the restaurant through Telegram while logging every order into Google Sheets for record keeping.

This eliminates manual order taking, reduces waiting time, and improves operational efficiency.

---

# 🎯 Business Problem

Traditional restaurant ordering often relies on waitstaff taking orders manually, which can result in:

- Longer waiting times
- Order mistakes
- Communication delays
- Manual record keeping
- Inefficient operations

TableFlow provides a contactless and automated ordering experience.
